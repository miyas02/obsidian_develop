# **コンセプト**

- 視聴者が必要な情報を閲覧できる
- 言論の視覚化
# 機能

- プレイヤーの投票入力
- 投票履歴の表示
    - 円グラフやラインチャートで視覚的にわかりやすく
    - 東京票や無効票なども表現したい
- 霊能者のCO状況の表示
    - UIに組み込み、視覚的にわかりやすく
- 発言メモ
- リアルタイムで必要なので、書きやすく
- AIによる要約機能
- 発言内容やタイミングを視覚的にわかりやすく
- 村作成機能
- ログイン機能

# 開発計画

開発をPhaseに分けて進める。

- **Phase 1**
    - バックグラウンドAPIの開発
    - メイン画面と投票画面の作成
- **Phase 2**
    - UIの調整
    - 録音機能やメイン画面の拡張
    - 常に外部要素を表示するサブ画面の追加
- **開発環境**
- MAUI + Razor (C#)
- Windowsネイティブアプリでプロトタイプを作成
- 完成後はWebアプリへ移行
# 技術仕様
- Blazor WASM (Webアプリケーション)
- 開発環境: **C# Blazor Visual Studio**
- Github Pages : Web Server,
