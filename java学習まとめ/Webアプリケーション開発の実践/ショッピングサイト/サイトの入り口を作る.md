このWebアプリケーションでは、コンテキストルートにウェルカムファイル(index.jsp)を置いて、以下のURLからアクセスするものとする

http://localhost:8080/book/chapter25/

メニュー部分をmenu.jspとして分離してインクルードするものとする。
こうしておくことで、ほかのページでもmenu.jspをインクルードするだけで、メニューが表示される。

### JSPファイルの作成

```java
index.jsp
<%@page contentType="text/html; charset=UTF-8" %>
<%@include file="../header.html" %>
<%@include file="menu.jsp" %>
includeディレクティブにより、menu.jspをインクルードしている

寿司オンラインショッピングにようこそ！

<%@include file="../footer.html" %>
```

```java
menu.jsp
<%@page pageEncoding="UTF-8" %>
pageディレクティブのpageEncoding属性を使って、JSPファイルの文字エンコーディングをUTF-8に指定する。
```
