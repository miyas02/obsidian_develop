## ショッピングサイトの構築

ショッピングサイトを開発するには、これまでに学んだ以下のような知識が必要

- サーブレットの作成
- JSPの作成
- サーブレットとJSPの連携
- データベースの操作
- セッションの利用
- javaBeansによるデータの管理
- JSPにおけるELやJSTLの利用
- デザインパターンの適用(DAO,MVC,Front,Controllerなど)
