### 引数

メソッドを呼び出すときに一緒に渡すと、メソッドの中でその値を利用できる
メソッドに引数を渡すには、まず引数を受け取れるメソッドを定義する
そのときに使う変数を仮引数

```java
public static void hello(String name //借り引数){
	System.out.println(name//引数「Bob」 + "さんこんにちは");
}
main {
	hello("Bob");//引数
}
```

### 戻り値

メソッドの中で「**return**」を使うと、retunの値をメソッドの呼び出しもとに返すことができる
戻り値のあるメソッドは**戻り値のデータ型**を指定する**(voidのとこ)**

```java
public static int add(int a,int b){
	retun a + b; //戻り値
}
```

```java
main {
	int total = add(7.5);
	System.out.println(total);
}
public static int add(int a,int b) {
	retun a + b;
}
```
