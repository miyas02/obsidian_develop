# 環境構築
### npmのインストール
```bash
# nodeインストール (未インストール時)
winget install OpenJS.NodeJS.LTS
# 実行環境を整えます。
npm install tailwindcss @tailwindcss/postcss postcss
```
nodeインストール後
gemini cliインストール
```
npx https://github.com/google-gemini/gemini-cli
```
gemini CLI インストール確認
```bash
gemini --version
```
gemini CLI 実行
```bash
gemini
```
