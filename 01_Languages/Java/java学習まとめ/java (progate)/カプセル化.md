フィールドとメソッドへのアクセスを制限すること
クラスの外部からアクセスきるのは**「public」**
アクセスできないのは**「private」**

### ゲッター

フィールドをprivateにしたうえで、クラス外から安全にフィールドの値を取得するために、フィールドの値を返すだけのメソッドを定義すること

```java
Main.java
class Main {
	public static void main(String[] args){
	Person person = new Person("John","Christopher",Smith,):
	System.out.println(person.getMiddleName());ゲッターを使ってアクセスする
	}
}
```

```java
Person.java
	class Person {
		private String middleName; privateフィールド,このままだとクラス外からアクセス不可
		public String getMiddleName() { クラス外からアクセス可能
			return this.middleName;
	}
}
```

### セッター

フィールドの値を変更するメソッド
mainクラスからPersonクラスのフィールドを変更したい

```java
セッターを使用しないエラーパターン
Main.java
	class Main {
		public static void main(String[] args){
			Person person = new Person("Kate","Jones");
			person.middleName = "Claire";エラー、privateなのでアクセスできない
```

```java
Main.java
	class Main{
		public static void main(String[] args){
			Person person = new Person("Kate","Jones");
			person.setMiddleName("Claire");
			System.out.println(person.getMiddleName());
	}
	}
```

```java
Person.java
	class Person {
		private String middleName;
		public void setMiddleName(String middleName){
			this.middleName = middleName; フィールドに値をセット
	}
}
```
