スコープとは名前の有効範囲のとこ。
プログラミングにおいては、変数名や関数名などを利用できる範囲のことをスコープとよぶ。

属性とは、サーブレット/JSPのデータ（オブジェクト）を保存するための仕組み。

> [!note]+ Attributeメソッド
> getAttributeメソッド`
> Object getAttribute(String name)
> `引数に指定した名前の属性から、データを取得します。戻り値がObject型であることに注意する。指定した属性名が存在しない場合はnullを返す。
> 
> setAttributeメソッド
> `void setAttribute (String name, Object object)`
> 第1引数で指定した名前の属性に、第2引数のデータを設定します。
> 
> removeAttributeメソッド
> `void remobeAttribute(String name)`
> 引数で指定した名前の属性を削除します。
> 
> getAttributeNamesメソッド
> `Enumeration<String>getAttribute( )`
> 属性名の一覧を種特区します。戻り値はgetParameterNamesメソッドと同じくEnumerationオブジェクトです。

**WEBアプリケーションでは、サーブレットがリクエストパラメータやデータベースの処理を行い、その結果をJSPに渡してレスポンスを出力するという方式をよく使う。その際結果を保持するためにリクエスト属性やセッション属性が使用される。ここではリクエスト属性を使う。
リクエスト属性には任意のオブジェクトを設定することができる。ここではサーブレットからリクエスト属性にBean(Productオブジェクト)を設定し、JSPから取得するプログラムを作成する。**
