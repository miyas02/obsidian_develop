# NET 全般の概念

- **dotNet**
    - **統合開発環境**であり、**クロスプラットフォーム**で利用されます。
- **dotNet SDK**
    - コンパイラや標準ライブラリを含む
    - Javaにおける JDK (Java Development Kit) に近いもの
    - JDK および JRE (Java Runtime Environment) のコンポーネントに相当する
- **CLR (Common Language Runtime)**
    - 共通言語ランタイム。
- **Core CLR**
    - **仮想実行環境**であり、Javaにおける **JVM (Java Virtual Machine)** に近いもの

# Web 関連技術

- **ASP.Net**
    - Web アプリケーションのための**フレームワーク**です。
- **Blazor**
    - **ASP.Net Core の一部**です。
    - `Blazor Server` と `Blazor WebAssembly` の形態があります。
    - Blazor は C# を使って UI を構築する技術です。
- **Razor**
    - UI 作成のための**テンプレート**です。
    - HTML の中に C# コードを埋め込む形式をとります。
