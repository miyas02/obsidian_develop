# VSCode ショートカット集
## 基本操作 (General)
- `Ctrl + Shift + P` : コマンドパレットを開く (すべての機能にアクセス)
- `Ctrl + ,` : 設定を開く
## 編集 (Editing)
- `Ctrl + /` : 行コメントの切り替え
- `Alt + ↑ / ↓` : 行を上下に移動
- `Shift + Alt + ↑ / ↓` : 行を上下に複製
- `Ctrl + D` : 選択した単語と同じ単語を次の出現箇所で選択 (マルチカーソル)
- `Ctrl + Alt + ↓ / ↑` : カーソルを上下に追加 (マルチカーソル)
- `Ctrl + Shift + K` : 行削除
## 検索・置換 (Search & Replace)
- `Ctrl + F` : ファイル内検索
- `Ctrl + H` : ファイル内置換
- `Ctrl + Shift + F` : フォルダ全体（ワークスペース）検索 (Grep)
## 折りたたみ (Folding)
- `Ctrl + K, Ctrl + 0` : すべて折りたたむ
- `Ctrl + K, Ctrl + J` : すべて展開する
- `Ctrl + K, Ctrl + 1` : レベル1で折りたたむ
