# Obsidian リンク作成方法
Obsidianの主要機能であるリンクの作成方法についてまとめます。
## 1. 内部リンク (Internal Links)
Vault内の他のファイル（ノート）へのリンクを作成します。Obsidianでは**Wiki Link**形式（二重ブラケット）が推奨されています。
- **構文**: `[[ファイル名]]`
- **使用例**: `[[Gitナレッジ]]`
  - 入力時にオートコンプリートが効きます。
  - リンクをクリックするとそのノートが開きます（`Ctrl` + クリック で別タブで開く）。
## 2. 外部リンク (External Links)
Webサイトなどの外部リソースへのリンクは、標準のMarkdown記法を使用します。
- **構文**: `[表示テキスト](URL)`
- **使用例**: `[Obsidian公式サイト](https://obsidian.md)`
## 3. リンクのエイリアス (表示名の変更)
リンク先のファイル名とは異なるテキストをリンクとして表示したい場合は、パイプ `|` を使用します。
- **構文**: `[[ファイル名|表示テキスト]]`
- **使用例**: `[[Gitナレッジ|Gitの使い方]]` -> 表示は「Gitの使い方」になりますが、リンク先は「Gitナレッジ」です。
## 4. 見出しへのリンク (Anchor Links)
ノート内の特定の「見出し」へ直接リンクすることも可能です。
- **構文**: `[[ファイル名#見出し名]]`
- **使用例**: `[[Gitナレッジ#基本コマンド]]`
## 5. ブロックへのリンク
段落（ブロック）単位でのリンクも可能です。
- **構文**: `[[ファイル名^ブロックID]]`
- **方法**: `[[ファイル名^]]` と入力すると、リンク可能なブロックの候補が表示されます。
